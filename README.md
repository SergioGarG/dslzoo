Robotics DSL Zoo
================

This is the repository that hosts the bib sources of the Robotics DSL Zoo. The
generated website is hosted in the gh-pages branch and is online here:
http://corlab.github.io/dslzoo/

How to contribute
-----------------

To contribute to the zoo, fork the repository, extend the dslzoo.bib file and
create a pull request. See also: http://corlab.github.io/dslzoo/contribute.html
